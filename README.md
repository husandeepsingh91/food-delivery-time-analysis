# Online Food Delivery Time Analysis 🍔📊

This project analyzes food delivery time patterns based on distance and restaurant performance.

## 🔍 Objectives:
- Analyze delivery time trends
- Compare restaurants
- Understand impact of distance

## 📊 Techniques Used:
- Data Analysis (Pandas)
- Data Visualization (Seaborn, Matplotlib)

## 📈 Output:
- Average delivery time
- Restaurant comparison graph
- Distance vs delivery time graph

## 🛠 Tools:
Python, Pandas, Seaborn, Matplotlib
